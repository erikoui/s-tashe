Class: PicturesRepository

db~PicturesRepository(db, pgp)

Manages the pictures table All the functions in the class are promises (from async) so you can call them with db.pictures.function(args).then(()=>{}).catch(()=>{}), or you can await them in async functions. It does not use tempalte strings for queries for security.
